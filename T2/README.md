# Task 02: Data Cleaning and Exploratory Data Analysis (EDA)

## Objective

To clean the dataset and analyze patterns using the Titanic dataset.

## Dataset

Titanic dataset containing passenger details like age, gender, class, and survival.

## Steps Performed

* Data loading and inspection
* Handling missing values
* Data cleaning
* Exploratory data analysis (EDA)
* Data visualization

## Tools Used

* Python
* Pandas
* Matplotlib

## Key Insights

* male passengers had a higher survival rate than females
* Passengers in higher classes had better survival chances
* Most passengers were between 20–40 years old

## Conclusion

EDA helped uncover patterns and relationships in the dataset, improving understanding of survival factors.
